## Preview the HTML File

To preview the webpage locally, you can use any of these methods:

### Quick Start (Python)
```bash
npm run preview
```
or
```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000/IronVector.html](http://localhost:8000/IronVector.html) in your browser.

### Alternative Methods
- **Node.js**: `npm run preview:node`
- **VS Code**: Use the "Live Server" extension
- **Direct**: Simply open `IronVector.html` in your browser
